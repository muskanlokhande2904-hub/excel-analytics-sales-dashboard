# Interactive Sales Dashboard using Microsoft Excel

## Overview
Built an interactive Excel dashboard to analyze regional sales performance, target achievement, and executive-wise KPIs.

## Features
- Regional sales analysis
- Executive-wise KPI tracking
- Target vs Actual comparison
- Interactive slicers
- Pivot Tables & Pivot Charts

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Form Controls
